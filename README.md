# Pluwfun 會員限定五月桃訂購網站第一版

這是第一版可操作原型，先用本機瀏覽器資料保存訂單與庫存，用來確認流程是否正確。

## 如何開始第一步

直接用瀏覽器打開：

```text
index.html
```

或在檔案總管中點兩下 `index.html`。

## 目前已完成

- 首頁主視覺與 Pluwfun 會員導流
- 1999 年度會員方案說明
- 五月桃規格選購
- 飛鼠幣折抵條件
- 自取 / 宅配單點 / 宅配多點
- 多點寄送數量分配與運費計算
- 桃子採購注意事項必勾
- 訂單送出後同步進入管理後台
- 庫存扣除
- 後台訂單狀態：新訂單、已收款、已派單、發貨中、完成配送、已分潤
- 農民分潤與平台收入試算
- 一鍵複製農民出貨明細

## 後台入口

在網站上方點：

```text
管理後台
```

或網址加上：

```text
#admin
```

## 下一階段

確認流程後，再把本機資料改接 Firebase：

- `orders` 訂單資料
- `products` 商品與庫存
- `shipments` 自取與宅配地址
- `settlements` 分潤資料
- 管理員登入與 Firestore 安全規則
